# Homebrew Tap for OxideKit

This tap contains Homebrew formulae for OxideKit projects.

## Installation

```bash
brew tap oxidekit/homebrew-tap
```

## Available Formulae

### OxidePM

Fast, modern process manager for Node.js and Rust applications.

```bash
brew install oxidekit/homebrew-tap/oxidepm
```

Or after tapping:

```bash
brew install oxidepm
```
